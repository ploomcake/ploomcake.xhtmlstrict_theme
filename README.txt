Introduction
============

This product is an xhtml strict template layer (with some patches) of ploomcake.theme
We focused on front-end facing views.

views tested:

standard plone view and portlets (without authoring)
tabs and slides views and portlets
quickupload portlet
action portlet
contentwellportlet
search
sharing
token role
flowsearch

NOTE: this is not a standalone package, thus you cannot install it individually.
Please visit http://www.ploomcake.org/en/download to get Ploomcake.

Ploomcake is a Plone distribution that allows you to create a ready-to-use Plone website, collaboration portal or
intranet, already configured with many addons.
